# Architectural Specification for Teacher Timetable PDF Export

## Objective

Design the timetable export so that the generated PDF is visually identical to a manually printed school timetable. The exported document must resemble a professionally formatted A4 landscape timetable exactly like the reference image.

The goal is not merely to display timetable data, but to recreate the entire printed layout including spacing, merged cells, vertical labels, break sections, headers, borders, typography, and alignment.

The exported PDF must look like it came from a school administration office.

---

# Page Setup

Paper Size

A4 Landscape

Margins

Top: 15mm

Bottom: 15mm

Left: 12mm

Right: 12mm

Use the full printable width.

Everything should be centered on the page.

---

# Overall Structure

The timetable consists of one large table.

Rows represent weekdays.

Columns represent lesson periods.

The entire table has a thin border surrounding it.

All cells have visible borders.

The structure should be:

```

| School Name                                                  |
|                     Teacher Name                             |
```

# Empty Cells

Empty lesson slots must still show borders.

Never collapse empty cells.

The printed timetable must preserve the grid.

---

# Alignment

Everything should align to a strict grid.

No uneven spacing.

All columns have fixed width.

Rows have equal height.

The timetable should look like it was created using Microsoft Word tables.

---

# Dynamic Data

The export should not hardcode lesson entries.

The system must dynamically fill:

Teacher Name

School Name

Subjects

Classes

Streams

Periods

Times

Weekdays

Empty periods

The layout must remain unchanged regardless of the data.

---

# PDF Quality

The exported PDF must be high resolution.

Suitable for direct printing.

No clipped text.

No overlapping text.

No automatic resizing that breaks alignment.

---

# Rendering Rules

Use merged cells for vertical break columns.

Support rotated text.

Maintain fixed table dimensions.

Prevent page breaks inside the timetable.

Everything must fit on one landscape A4 page.
